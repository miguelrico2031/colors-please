# Colors, Please
¡Bienvenido a Colors, Please: EyeDropper Simulator!

¡Ponte en la piel de Dwight Dropper, un simpático cuentagotas que lucha por sobrevivir en un mundo donde cada color cuenta! En este creativo y desafiante juego, deberás poner a prueba tu habilidad para identificar y seleccionar colores, enfrentándote a una variedad de minijuegos únicos y llenos de personalidad. Cada acierto te acerca a pagar el alquiler, mantenerte a flote y descubrir los secretos que esconde la historia de Dwight.
